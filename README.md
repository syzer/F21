## SETUP
### Flask server

1. Go to the discounts-server directory
```bash
$ cd discounts_server
```
2. Export the `FLASK_APP` environment variable
```bash
$ export FLASK_APP=server.py 
```
3. Run the flask server :ghost:
```bash
$ flask run 
```
